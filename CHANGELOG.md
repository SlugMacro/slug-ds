# slug-ds

## 0.1.0

### Minor Changes

- Initial release of slug-ds design system.

  29 components: Button, Text, Badge, Input, Card, Heading, Alert, Dialog, DropdownMenu, Tooltip, Toast, Tabs, Accordion, Popover, Label, Textarea, Checkbox, Radio, Switch, Select, Box, Stack, Grid, Separator, Container, Spinner, Avatar, IconButton.

  Features:
  - Full design token system (colors, spacing, typography, shadows, radii, breakpoints)
  - Light and dark theme support via CSS custom properties
  - CVA-powered variant system for all components
  - Radix UI primitives for accessible interactive components
  - Polymorphic `as` prop on layout components
  - Compound component API (Card, Dialog, Tabs, Accordion, Alert, etc.)
  - WCAG 2.1 AA compliant with jest-axe tested accessibility
  - ESM + CJS dual output with TypeScript declarations
